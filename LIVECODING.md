# Consumo de API

## Objetivo

Accede a la API de Pokemon y muestra los pokemones en una lista paginada, muestra nombre e imagen, al seleccionar un pokemon muestra sus detalles en un modal.

## Detalles del entregable
- Tiempo estimado de desarrollo: 15 minutos
- Prioriza el funcionamiento sobre el diseño

## Información adicional

### Documentación de la API
- https://pokeapi.co/docs/v2#resource-listspagination-section (Uso de paginación)
- https://pokeapi.co/docs/v2#pokemon (Obtener información de un pokemon)
